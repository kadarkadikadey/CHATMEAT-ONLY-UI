# ChatMeat 

**ChatMeat** is a high-performance, visually immersive social networking application built with **Flutter**. It combines a sleek, modern UI with a multi-step user onboarding process and a robust communication interface.

---

## 🚀 App Flow & Features

Based on the latest UI walkthrough, ChatMeat includes:

### 1. Onboarding & Security
* **Splash & Welcome:** High-quality visual entry point featuring anime-inspired aesthetics (Raiden Shogun/Yae Miko).
* **Dual Auth System:** Seamless Login and Sign-in toggles for a better user experience.
* **Secure OTP:** Integrated verification flow including "Send OTP" and password creation screens.

### 2. User Personalization
* **Detailed Profiling:** Modular sections for Name, Username, Bio, and "About" info.
* **Media Identity:** Dedicated screens for setting up circular Profile Pictures and rectangular Banner Pictures.

### 3. Core Interface
* **Main Hub:** Tabbed navigation categorized by **Chat**, **Group**, and **Store**.
* **Chat Window:** Clean messaging UI with real-time status and interactive input.
* **Profile Grid:** A custom 3-column **Instagram-style** media gallery for post discovery.

---

## 🎨 Visual Identity
The app follows a consistent **Blue and Violet** color scheme, designed for a modern and premium feel. All icons are integrated using **FontAwesome** to ensure a high-quality social media aesthetic.

---

## 🛠️ Technical Stack

* **Framework:** [Flutter](https://flutter.dev/)
* **Language:** [Dart](https://dart.dev/)
* **Icons:** FontAwesome & Material Icons
* **Layout:** Custom Sliver-based Grids and TabBar navigation

## 💻 Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/yourusername/chatmeat.git](https://github.com/yourusername/chatmeat.git)
    ```

2.  **Navigate to the project folder:**
    ```bash
    cd chatmeat
    ```

3.  **Install dependencies:**
    ```bash
    flutter pub get
    ```

4.  **Run the application:**
    ```bash
    flutter run
    ```
